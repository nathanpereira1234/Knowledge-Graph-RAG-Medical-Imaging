# Results

Committed on purpose. A reviewer should be able to read the numbers without cloning, installing, or running anything.

| File | What it is |
|---|---|
| `metrics.json` | All metrics, all ablations. Generated by `scripts/run_eval.py --ablations`. |
| `resolved_config.yaml` | The exact fully-merged config that produced `metrics.json`. |
| `predictions.jsonl` | Per-sample predictions with evidence traces. |
| `figures/` | Plots for the report. |

## Regenerating

```bash
python scripts/run_eval.py --config configs/eval.yaml --ablations
```

Never hand-edit `metrics.json`. If it disagrees with the README table, rerun the script and update the README from its output — not the other way around.
